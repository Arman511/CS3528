Add User

To add a user, you need to provide the following details:

  • Name
  • Email
  • Password
  • Confirm Password
There can not be 2 users with the same email address.

Delete User

To delete a user, navigate to the user management section and select the user you wish to delete.

Update User

To update user details, navigate to the user management section and select the user you wish to update.

There can not be 2 users with the same email address.

Change Password

To change a user's password, navigate to the user management section and select the user whose password you wish to change.

You will be asked for their new password and confirm password

Config

The configuration options include:

  • Max number of skills a student can input
  • Min number of rankings a student must provide for opportunities

Change the details and press submit to save the changes.
