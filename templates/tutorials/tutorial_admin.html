Login

Enter the email and password associated with your account. After successful login you will be redirected to the homepage.

Navigation

Moving between features is achieved by using the navigation bar found just below the website title. By selecting elements within this bar you will be redirected to other components of the website.

Home

This will take you back to the main home page.




Students

When selected, you will be presented with a drop down menu, containing multiple options:

  • Search
    • Where you can locate and search specific students, to retrieve/edit their information.

  • Upload Students
    • Here you can upload a spreadsheet to automatically upload students and their information, instead of manual entry.

  • Add Students
    • For single manual entry of a students information.

  • Download All Students
    • Downloads all student information stored on the SkillPilot platform in an excel spreadsheet.

  • Delete All Students
    • Removes all student information from the SkillPilot platform.

      This is irreversible.

Searching Opportunities




The SkillPilot platform provides a mechanism to granularly find specific opportunities. Search results include general information about a given opportunity. From the results we can also edit information about an opportunity, or remove them entirely.

  • Searching
    • We can locate oppertunities using any of the following criteria:
    • Title
    • Company
    • Ranking Status

  • Granular Search
    • We can use more fine grained search to locate oppertunities, by expanding the 'Show More' button.
    • Description
    • Location
    • Course
    • Modules
    • Min Placements
    • Max Placements
    • Placement Duration

Search Results


From the information entered previously, if any matching opportunities are found, they will be displayed below the search card of the page, under 'Opportunities'. The search results include the following information for each opportunity:

  • Details
    • Title
    • Description
    • Company Name
    • Company URL
    • Location
    • Courses Required
    • Modules Required
    • Spots Available
    • Placement Duration
    • Ranked Status
    • Actions

Any of the information listed above can be edited, by selecting the 'Update' button, in the furthest right column titled 'Actions'.

Search Attempted Skills






Attempted skills are skills that have been uploaded by students that require your aproval. During the application process students are able to add relevant skills which are not currently on the SkillPilot platform.

Any skill a student uploads requires aproval to be added to the platform. You are able to provide approval here, the Search Attempted Skills section. From here you can either approve, update or reject skills.

When you first arrive on this page, below under the 'Attempted Skills' section, will contain all skills pending aproval. All skills here will contain the; Name of the skill, description, frequency of the skill in applications and a set of actions applicable. These are all available in the 'Attempted Skills' section of the page, in their respective columns.

  • Approving Skills
    • 1. Search or manually locate skill
    • 2. Go to action column on far right
    • 3. Select the 'Approve' button

  • Rejecting Skills
    • 1. Search or manually locate skill
    • 2. Go to action column on far right
    • 3. Select the 'Reject' button

  • Updating Skills

    • If you wish to change details of a pending skill:

    • 1. Search or manually locate skill
    • 2. Go to action column on far right
    • 3. Select the 'Update' button
    • 4. Change either 'Skill Name' and/or 'Skill Description'
    • 5. Select the 'Update Skill' button to save changes

Matching


The matching process is designed to be fully automatic, without many input during its processing.

  • How Does It Work?
    • The placement team add a specific opportinity on behalf of a business/institution.
    • Students enter their relevant such as, skills, modules, courses, etc
    • Students then rank order the placements in terms of preference
    • Employers rank order students based on preference
    • The SkillPilot algorithm matches students to employers based of preference
    • Students will then be notified of placement success or failure

  • Rejecting Skills
    • 1. Search or manually locate skill
    • 2. Go to action column on far right
    • 3. Select the 'Reject' button

  • Updating Skills

    • If you wish to change details of a pending skill:

    • 1. Search or manually locate skill
    • 2. Go to action column on far right
    • 3. Select the 'Update' button
    • 4. Change either 'Skill Name' and/or 'Skill Description'
    • 5. Select the 'Update Skill' button to save changes

Problems

The matching process employed by the SkillsPilot platform is used to ensure that all applicants are paired with employers based off of preference. with minimal matched pairs.

This process works behind the scenes in the SkillsPilot platform, requiring no input from the university placement team.

Students may not be matched with employers for reasons of spaces, and/or ranking criteria, whereby student or employer ranked one another lower than other potential students. In the case of unmatched students, a card section on the matching page is viewable, containing all unmatched students and its corresponding reason, when this occurs, students can be deleted from the SkillsPilot platform.

When students have successfully matched with a placement provider, under the matching section, contains a list of all students and their matched placement providers. After matching email notification can be provided informing applicants of success, by selecting the 'Send All Emails' at the top of the webpage. Conversely, individual emails can be sent by selecting the 'Send Email' button next to the given student.
