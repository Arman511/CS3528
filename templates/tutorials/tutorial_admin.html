Login

Enter the email and password associated with your account. After successful login you will be redirected to the homepage.

Navigation

Moving between features is achieved by using the navigation bar found just below the website title. By selecting elements within this bar you will be redirected to other components of the website.

Search Student

Upon selecting search student in the below subcard we can use information such student names, ID numbers and course information to locate group of specific students. By locating students we are able to change their SkillsPilot details by selecting the 'Update' button next to the corresponding student.

Within the update card we can view and edit their personal and university information. Ensure that the 'Update' button is pressed to ensure changes made persist.

Upload Students

By selecting the 'Upload Students' subheading under the navigation bar, you can upload an excel formatted spreadsheet. Once uploaded the student information and details present will be uploaded and saved to the database. Providing a more efficient method than manually entering student information.

Add Student

Conversely, instead of bulk adding students details in bulk. We can individually enter information on a per student basis.

On selection, you'll be presended with a card view. Within this view we can enter details about specific students.
  • Student Number
  • First Name
  • Last Name
  • Email Address
  • Student Subscribed Courses
  • Student Modules Undertaken
  • Additional Comments

Upon entering student details, by selecting the 'Update' button, all details entered will be saved to the database, as a new student.

Delete All Students

This will remove all saved student details within the database.

This is irreversible.

Download All Student

By selecting this section, an excel file will download containing the information of all saved students. In the format of:

  • First Name
  • Last Name
  • Email Address
  • Student Number
  • Course Code
  • Selected Skills
  • Additional Comments
  • Placement Duration

Search Opportunities

This section is used to locate specific placement opportunities via placement information. Placements can be located by using any of the following information:

  • Title
  • Company Name
  • Description
  • Location
  • Placement Description
  • Course Prerequisites
  • Required Modules
  • Minimum Placement Positions
  • Maximum Placement Positions
  • Placement Duration
After entering any information in the card view, below will be a grid of all the placements matching the criteria. Each placement opportunity is displayed in the format of:

  • Placement Title
  • Description
  • Company Name of Placement Provider
  • URL to Placement Provider
  • Placement Location
  • Prerequisite Course Codes
  • Required Modules
  • Amount of Available Placements
  • Duration of Placement
To the right of each individual placement are two buttons:
  • Update
  • Delete
Update: Upon selection, you will be forwarded to a card view containing the information of the given placement. From here you can update relevent details, such as:
  • Unique ID of Placement
  • Placement Title
  • Description
  • Required Course Codes
  • Placement Provider Website
  • Placement Location
  • Available Placement Positions
  • Duration
  • Required Modules
All of the above information is editable, except the unique ID of the placement. Ensure after changes to select the 'Add/Update' button to save any changes.

Upload Opportunities

From here instead of manually adding each placement, we can upload a spreadsheet that contains the information required to add a placement.
Select a spreadsheet file, from the choose file button. Upon selection, press the 'Upload' button to add the information to the database.
Additionally, you can download a template of the spreadsheet file to get started with.

Search Employers

The search card for employers offers two mechanisms to filter:

  • Company Name
  • Company Email

Upon entering this information, below the search card will contain matches of the query.

Upload Employer

By selecting the 'Choose File' button, you can select an excel spreadsheet containing information of multiple employers, instead of manual entry. After a file has been chosen, select the 'Upload' button to upload the spreadsheet data.

Additionally, a template spreadsheet can be downloaded by selecting the 'Download Template' button.

Add Employer

To add single employers at a time. Enter the company name and corresponding email address, once entered press the 'Add Employer' box to save details. For the employer to login to the SkillPilot portal, they will use these details. The login process for employers is by employers entering their email address in the employers login portal, where a one time passcode will be sent to their address, where upon entering this code access is granted. To receive the code ensure employers have checked their spam folder, adding the address as a contact if so.

Delete Employer

Removes all saved employers and their details

This is irreversible.

Download All Employer

Upon selection, it will automatically download a spreadsheet containing infortmation stored of all saved employers.

Search Courses

Presented to you will be a card view titled 'Search Course', the card will contain 3 input boxes.

  • Search Course Name – e.g. 'Computer Science'
  • Search Course ID – e.g. 'F421'
  • Search Course Description – e.g. 'Artificial Intelligence'

After entering information in one of the input boxes, the below list of courses will dynamically change to find any matching criteria. From within this list, we can complete 2 actions.

  • Update
  • Delete

The update button takes us to a new view, where you can change any details about the specific course:

  • Course ID
  • Course Name
  • Course Description

When entering course information, please note that information should follow the same format as the university provides.

The delete button removes the given course and its information from the database.

Upload Courses

From here, you can upload a formatted spreadsheet to bulk upload courses to SkillPilot, by clicking the 'Choose File' and selecting an appropriate spreadsheet file then selecting the 'Upload' button, all courses in the spreadsheet will be uploaded to SkillPilot.

A template is availble to download by selecting the 'Download Template' button, that can be added with course information and later uploaded using the above process.

Add Course

Select this option to manually enter a given course into SkillPilot.

Here you will be welcomed by a card view, containing 3 text boxes:

  • Course ID
  • Course Name
  • Course Description

To successfully enter a course into SkillPilot, you must provide information into all 3 fields. Once entered, press the 'Add Course' button to save the course to SkillPilot.

Please note: The course details should accurately follow the same format that the university provides.

Delete All Courses

Removes any & all saved courses and their information from the SkillPilot platform.

This is irreversible.

Download All Courses

Upon pressing this button, a download will automatically start. This is a copy of the database contents of all the courses and their information in a spreadsheet format.

Search Skill

Upon selecting the 'Skills' section of the menu bar, you will be taken to the seach skills screen. From here you will see two input boxes contained within a card view. By entering information in the corresponding input boxes, it allows us to locate specific skills.

If you are trying to locate a specific skill that is not showing during search, remember skills not present must be entered in the 'Upload Skills' section of the skills page.

  • Search Skill – e.g. 'Python'
  • Search Skill Description – e.g. 'Programming with python'

After entering information in one of the input boxes, the below list of courses will dynamically change to find any matching criteria. From within this list, we can complete 2 actions.

  • Update
  • Delete

The update button takes us to a new view, where you can change any details about the specific course:

  • Skill Name
  • Skill Description

Please note, that when altering or adding new courses, please ensure that a consistent format is followed.

The delete button removes the given course and its information from the database.

Search Attempted

When students apply via the SkillsPilot platform, during the application process they will have the opportunity to add their applicable skills. During this process, students can add skills that are not found within the current list of skills.

When this occurs, all requested skills will be viewable from this page. A list of all requested skills is viewable in the card view titled 'Attempted Skills'

New skills can be added at the discretion of the placement team, where they have the option to approve or reject requested skills, or with the 'Update' button, alter either the skill name or description of the requested skill.

In the card above, titled 'Search Opportunities' we can search and filter the attempted skills, by either entering the:

  • Skill Name
  • Skill Description
  • Min Usage - (How little skill is found in other applications)
  • Max Usage - (How often skill is found in other applications)

The below list will dynamically adjust, showing any matching skills in the below 'Attempted Skills' card.

A template is availble to download by selecting the 'Download Template' button, that can be added with course information and later uploaded using the above process.

Upload Skills

This option will allow the placement team to upload a spreadsheet containing bulk skills, instead of manual entry to the SkillsPilot platform. To upload a spreadsheet, select the 'Choose File' button, select the appropriate file, then press the 'Upload' button.

A template is available to download that is predefined in the required format for upload.

Add Skill

Used for single entry of skills.

In the webpage, presented to you will be a card view titled 'Add Skill' within the card will contain 2 input boxes:

  • Skill Name
  • Skill Description

To enter a skill to the SkillsPilot platform, enter the corresponding information into each text box, then selecting the 'Add Skill' button. For the skill to be saved to SKillsPilot.

Delete All Skills

By selecting this button, all skills saved on the SkillsPilot platform will be permenantly deleted.

This is irreversible.

Delete All Attempted Skills

By selecting this button, all skills requested to be added to the SkillsPilot platform, but applying students, will be permenantly deleted.

This is irreversible.

Download All Skills

Download All Attempted Skills

Matching

The matching process employed by the SkillsPilot platform is used to ensure that all applicants are paired with employers based off of preference. with minimal matched pairs.

This process works behind the scenes in the SkillsPilot platform, requiring no input from the university placement team.

Students may not be matched with employers for reasons of spaces, and/or ranking criteria, whereby student or employer ranked one another lower than other potential students. In the case of unmatched students, a card section on the matching page is viewable, containing all unmatched students and its corresponding reason, when this occurs, students can be deleted from the SkillsPilot platform.

When students have successfully matched with a placement provider, under the matching section, contains a list of all students and their matched placement providers. After matching email notification can be provided informing applicants of success, by selecting the 'Send All Emails' at the top of the webpage. Conversely, individual emails can be sent by selecting the 'Send Email' button next to the given student.
